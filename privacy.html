Política de Privacidad

Última actualización: 12 de junio de 2026

1. Información que recopilamos

R7 Kryon recopila la siguiente información para funcionar correctamente:

  • IDs de usuario de Discord (para el sistema de economía y sugerencias)
  • IDs de servidor de Discord (para configurar comandos por servidor)
  • IDs de canales de Discord (para enviar mensajes de sugerencias y postulaciones)
  • Mensajes enviados directamente al bot o que mencionan al bot (para la IA)
  • Respuestas de formularios enviadas a través de webhooks

2. Cómo usamos la información

La información recopilada se usa exclusivamente para:

  • Gestionar el sistema de economía (R7 Coins, inventario, banco)
  • Procesar sugerencias y postulaciones de los miembros
  • Responder consultas a través de la IA R7 Kryon
  • Configurar el bot correctamente en cada servidor

3. Almacenamiento de datos

Los datos se almacenan en archivos JSON en el servidor donde corre el bot. No se comparten con terceros ni se venden bajo ninguna circunstancia.

Los datos de conversación con la IA se mantienen temporalmente en memoria (máximo 10 mensajes por usuario) y no se almacenan de forma permanente.

4. Retención de datos

Los datos del usuario se conservan mientras el bot esté activo en el servidor. Si el bot es expulsado de un servidor, los datos asociados a ese servidor pueden ser eliminados.

5. Tus derechos

Tienes derecho a:

  • Solicitar la eliminación de tus datos
  • Saber qué datos tenemos sobre ti
  • Solicitar la corrección de datos incorrectos

6. Servicios de terceros

R7 Kryon utiliza los siguientes servicios de terceros:

  • Discord API — para funcionar como bot de Discord
  • Groq API — para procesar mensajes de IA (los mensajes se envían a Groq para generar respuestas)

7. Cambios a esta política

Podemos actualizar esta política en cualquier momento. Los cambios serán efectivos al publicarse en esta página.
